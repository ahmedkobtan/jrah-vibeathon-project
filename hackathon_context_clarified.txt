üõ†Ô∏è Hackathon-Friendly Agent Design
You don‚Äôt need a full LLM-based agent. A rule-based or plugin-style parser manager works well:

python
class ParserAgent:
    def __init__(self):
        self.parsers = {
            "json": JSONParser(),
            "csv": CSVParser(),
            "zip": ZipParser(),
        }

    def detect_format(self, file_path):
        if file_path.endswith(".json"):
            return "json"
        elif file_path.endswith(".csv"):
            return "csv"
        elif file_path.endswith(".zip"):
            return "zip"
        else:
            raise ValueError("Unsupported format")

    def parse(self, file_path):
        fmt = self.detect_format(file_path)
        return self.parsers[fmt].parse(file_path)
Each parser class (e.g., JSONParser) handles its own quirks and normalization logic.
